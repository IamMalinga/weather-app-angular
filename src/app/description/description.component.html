Project Details
This is a simple weather app designed using Angular, utilizing the Open Cloud API to fetch weather data. The app's straightforward design ensures a user-friendly experience while offering accurate and up-to-date weather information. By seamlessly integrating with the Open Cloud API, users can access real-time weather data, enhancing the app's functionality. The clear and concise structure, coupled with the use of Angular, makes this weather app an efficient and user-centric tool for staying informed about current weather conditions.

The API keys can be described as follows:

coord coord.lon Longitude of the location coord.lat Latitude of the location

weather.main Group of weather parameters (Rain, Snow, Clouds etc.)

main.temp Temperature. Unit Default: Kelvin, Metric: Celsius, Imperial: Fahrenheit

main.pressure Atmospheric pressure on the sea level, hPa

main.humidity Humidity, %

main.temp_min Minimum temperature at the moment. This is minimal currently observed temperature (within large megalopolises and urban areas).

main.temp_max Maximum temperature at the moment. This is maximal currently observed temperature (within large megalopolises and urban areas).

main.sea_level Atmospheric pressure on the sea level, hPa

main.grnd_level Atmospheric pressure on the ground level, hPa

wind.speed Wind speed. Unit Default: meter/sec, Metric: meter/sec, Imperial: miles/hour

clouds.all Cloudiness, %

rain.1h (where available) Rain volume for the last 1 hour, mm. Please note that only mm as units of measurement are available for this parameter

snow.1h (where available) Snow volume for the last 1 hour, mm. Please note that only mm as units of measurement are available for this parameter

sys.country Country code (GB, JP etc.)
